# Marketing Skills for AI Agents

## Summary

A curated GitHub repository of 40+ markdown-based skill files designed for AI coding agents (Claude Code, Cursor, Windsurf, etc.). Each skill gives an AI agent specialized knowledge and workflows for a specific marketing task. A foundational `product-marketing-context` skill acts as the knowledge base that all other skills reference first, ensuring consistent product/audience/positioning context.

Licensed MIT. Created by [[corey-haines]].

## Key Takeaways

- Skills are plain markdown files — portable, versionable, agent-agnostic
- The `product-marketing-context` skill is the anchor: it holds your product context and all other skills pull from it
- 9 skill categories covering the full marketing stack (see below)
- Installable via CLI (`npx skills`), Claude Code plugin, git clone, submodule, or fork

# How To Get The Most Out of Claude for Product Marketing Managers

## Summary

LinkedIn post and accompanying infographic by [[hattie-the-pmm]]. Core argument: prompting is the weakest way to use Claude — the gap between casual and power users is system-level setup, not prompting skill.

"Most PMMs are still just chatting with Claude. The power users are building systems with it."

Primary deliverable is a reference table mapping 10 Claude products to PMM use cases, with prompt examples for each, plus three advanced hacks. Published at productmarketers.com.

---

## Core Thesis

The leverage gap isn't effort or prompting sophistication. It's knowledge of system architecture: Projects for context persistence, Skills for standing briefs, Cowork for local file access, Connectors for tool integrations. Each layer multiplies leverage; each layer most PMMs skip.

---

## Claude Tool Reference (Infographic)

| Tool | What It Is | PMM Use |
|---|---|---|
| **Claude Chat** | Always-on strategist | Positioning docs, stakeholder prep, stress-test messaging. Use Opus 4.6 + Extended Thinking for strategy. |
| **Claude Projects** | Saved workspaces | One project per product line. Upload messaging framework, personas, competitive intel. No re-briefing. |
| **Claude Skills** | Standing brief | Build a Skill for writing style, brief format, or battle card structure. Claude follows it automatically. |
| **Claude Cowork** | Desktop app; reads local files | Point at folders of call recordings, analyst reports, win/loss data. Creates real documents. |
| **Claude Connectors** | Integrations (Slack, Drive, Notion, 50+) | Pull Slack threads, search Drive, reference Notion roadmap — without copy-pasting. |
| **Claude Research** | Deep reasoning + web search | Competitive landscape, analyst positioning, category narrative, market sizing. Use when citation matters. |
| **Claude Charts** | Interactive visuals in chat | Positioning maps, funnel visuals, customer journey diagrams, category comparison charts. |
| **Claude Excel** | AI inside spreadsheet | Clean survey data, pipeline influence models, explain formulas in plain English for Sales/Finance. |
| **Claude Plugins** | Pre-built skill packs | Sales enablement, SEO, finance. Install and Claude adds slash commands for that domain. |
| **Claude Code** | Reads data files, runs scripts | Product usage CSV → cohort breakdowns, feature adoption, pipeline influence reports. |

## Advanced Hacks

1. **Give Claude your stakeholder map.** Before any strategy task, tell Claude who the audience is, what they care about, and how they like to hear things.
2. **Make Claude pressure-test your messaging.** End any positioning prompt with: "Now play a sceptical Sales rep who thinks PMM is too fluffy. What's their first objection?"
3. **Set up Global Instructions for your role.** Settings > Profile. Write your product category, ICP, and tone of voice. Claude reads this before every conversation.
